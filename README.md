# Radio Player

## Introduction

Radio Player allows you to switch between two free radio stations : Radio 357 and Radio Nowy Åšwiat. You can find Radio Player demo [here](https://mtkuchta.github.io/radio-player/).

![Radio Player view](./src/assets/images/radio_player.jpg)

## Technologies

- React 17.0.1,
- HowlerJS 2.2.1,
- GSAP 3.6.0

## Setup

To run this project, install it locally using npm:

$ cd ../lorem
$ npm install
$ npm start
